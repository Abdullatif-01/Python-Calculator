# Python GUI Calculator

A simple and user-friendly calculator built with Python using Tkinter.
It provides a graphical interface for performing basic arithmetic operations.

## Screenshot

![Calculator Screenshot](screenshot.png)

## Features

* Addition, subtraction, multiplication, and division
* Percentage (%) support
* Clear button to reset calculations
* Error handling for invalid inputs
* Interactive button-based GUI

## Technologies Used

* Python
* Tkinter (built-in Python GUI library)

## How It Works

* The calculator uses a Tkinter window with buttons arranged in a grid.
* User input is displayed in an entry box.
* When "=" is pressed, the expression is evaluated and the result is shown.
* Invalid expressions return an "ERROR!" message.

## How to Run

1. Make sure Python is installed on your system.
2. Download or clone this repository.
3. Run the file:

```bash
python calculator.py
```

## Project Structure

* `calculator.py` → Main file containing the calculator logic and GUI
